run(simulations=None, clean=True, multithread=True)[source]#

Run simulations

Parameters

  • simulations – List of simulation names to run, if None runs all simulations, by default None.

  • optional – List of simulation names to run, if None runs all simulations, by default None.

  • clean – If True remove existing database for the file before running, by default True

  • optional – If True remove existing database for the file before running, by default True

  • multithread – If True APSIM uses multiple threads, by default True

  • optional – If True APSIM uses multiple threads, by default True
